With:
```js
function attachGridEvents() {
  $gridContainer.addEventListener('click', function (e) {
    /* chat button branch MUST come first — stopPropagation prevents card-open from firing */
    var chatBtn = e.target.closest('.card-chat-btn');
    if (chatBtn) {
      e.stopPropagation();
      var handle = chatBtn.dataset.handle;
      if (handle) {
        track('feed_card_click_chat', { persona_handle: handle });
        var url = 'https://t.me/' + encodeURIComponent(handle);
        if (tg && tg.openTelegramLink) {
          tg.openTelegramLink(url);
        } else {
          window.open(url, '_blank');
        }
      }
      return;
    }
    /* card body opens profile */
    var card = e.target.closest('.model-card');
    if (card && card.dataset.username) openProfile(card.dataset.username);
  });
}
```

Critical ordering: `.card-chat-btn` branch FIRST. `e.target.closest('.model-card')` would also match when the user taps the chat button (because `.card-chat-btn` is inside `.model-card`). Checking chat button first and calling `return` prevents the card-open branch from executing. `e.stopPropagation()` stops upward bubbling (not needed here since both branches are on the same delegated listener, but included for clarity and future-proofing).

`track()` is called BEFORE `tg.openTelegramLink()` (TRACK-07: event dispatched before Mini App may close).
