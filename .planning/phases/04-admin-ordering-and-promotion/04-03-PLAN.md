**3. Add observePromotedCards() function** after the renderGrid function definition. Call it at the end of renderGrid():

Place this function after renderGrid:
```javascript
function observePromotedCards() {
  if (!('IntersectionObserver' in window)) return;
  var cards = document.querySelectorAll('.model-card.promoted');
  if (cards.length === 0) return;
  if (window._promoObserver) window._promoObserver.disconnect();
  window._promoObserver = new IntersectionObserver(function (entries) {
    entries.forEach(function (entry) {
      entry.target.classList.toggle('offscreen', !entry.isIntersecting);
    });
  }, { threshold: 0 });
  cards.forEach(function (card) { window._promoObserver.observe(card); });
}
```

Then at the very end of the `renderGrid` function body (after the closing of the `$gridContainer.innerHTML = ...` assignment, before the function's closing brace), add:
```javascript
observePromotedCards();
```

**Important implementation notes:**
- `will-change: transform, opacity` ONLY on promoted cards, not all cards (per Research anti-pattern guidance)
- `border-radius: 18px` on ::before is 16px (card) + 2px (inset offset) to visually align
- Star character `\2605` avoids needing an SVG data URL or external asset
- `z-index: -1` on ::before places the glow behind card content
- `z-index: 2` on ::after places the star above the card overlay
- `pointer-events: none` on ::after prevents the star from blocking chat button clicks
- `window._promoObserver` stored for cleanup on re-render (same pattern as SortableJS instance)
